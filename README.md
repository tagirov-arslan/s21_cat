# s21_cat

## Описание

` s21_cat ` — это учебная реализация стандартной Unix-утилиты `cat`, написанная на языке C.

Программа читает содержимое одного или нескольких файлов и выводит его в стандартный поток вывода.  
Также поддерживаются дополнительные флаги для нумерации строк, отображения непечатаемых символов, табуляции и сжатия пустых строк.

---

## Поддерживаемые флаги

Утилита поддерживает следующие параметры:

| Флаг | Длинный вариант | Описание |
|------|------------------|----------|
| `-b` | `--number-nonblank` | нумерует только непустые строки |
| `-n` | `--number` | нумерует все строки |
| `-s` | `--squeeze-blank` | сжимает несколько подряд идущих пустых строк в одну |
| `-e` | — | эквивалент `-vE`: показывает непечатаемые символы и символ `$` в конце строки |
| `-E` | — | показывает `$` в конце каждой строки |
| `-t` | — | эквивалент `-vT`: показывает непечатаемые символы и заменяет табуляцию на `^I` |
| `-T` | — | показывает символ табуляции как `^I` |
| `-v` | — | показывает непечатаемые символы, кроме `\\n` и `\\t` |

---

## Приоритет флагов

Если одновременно используются флаги `-b` и `-n`, приоритет имеет `-b`.

То есть:
- `-b` нумерует только непустые строки
- `-n` в этом случае отключается

## Description

` s21_pong ` — это консольная игра **Ping Pong**, написанная на языке C с использованием библиотеки `ncurses`.

В игре участвуют два игрока, которые управляют ракетками с клавиатуры.  
Мяч движется по игровому полю, отскакивает от верхней и нижней границ, а также от ракеток.  
Если один из игроков пропускает мяч, сопернику засчитывается очко.

Игра продолжается до тех пор, пока один из игроков не наберёт **11 очков**.

---

## Controls

### Left player
| Key | Action |
|-----|--------|
| `s` | move up |
| `x` | move down |

### Right player
| Key | Action |
|-----|--------|
| `k` | move up |
| `m` | move down |

### General
| Key | Action |
|-----|--------|
| `q` | quit game |

---

## Rules

- Игра начинается с мячом в центре поля
- Мяч движется по диагонали
- При столкновении с верхней или нижней границей он меняет направление
- При столкновении с ракеткой мяч отскакивает назад
- Если мяч достигает левой или правой границы, засчитывается гол
- После гола:
  - мяч возвращается в центр
  - ракетки сбрасываются в начальное положение
  - направление мяча меняется
- Побеждает игрок, первым набравший **11 очков**
